❓ FAQ

Q. 봇이 감지를 안 해요
① 크롬에 IMI BOT 확장프로그램이 설치·활성화 되어 있는지 확인
② 모니터링 창에서 규칙에 체크가 되어 있는지 확인
③ 봇 상태가 ● 감시 중 인지 확인
④ 확장프로그램을 삭제 후 재설치해 보세요
Q. 알림 팝업이 안 떠요
Windows 설정 → 알림 및 작업 → Google Chrome → 알림 허용 확인
또는 크롬 주소창 자물쇠 아이콘 → 알림 허용
Q. 실수로 필터 제외했어요
📋 로그 → 3 🚫 필터제외 탭 → 해당 항목의 ↩ 제외 취소 버튼 클릭
Q. 비밀번호를 잊었어요
로그인 화면 → 비밀번호 찾기 → 이름·사번 입력 → 임시 비밀번호 발급
또는 관리자에게 비밀번호 초기화 요청
Q. 키워드에 콤마(,)가 포함된 단어를 감지하고 싶어요
콤마는 키워드 구분자로 사용되므로 키워드 자체에는 넣을 수 없습니다.
대신 콤마 없이 9검67셋으로 등록하면 9검6,7셋 형태도 자동으로 감지됩니다.
Q. 프로그램 업데이트 후 적용이 안 돼요
크롬 확장프로그램 관리 → IMI BOT 삭제 → 다시 설치 (리로드 아님, 반드시 삭제 후 재설치)
