📅 직원 스케줄 표

메인 화면 상단 헤더 가운데에 표시된 날짜/시간 텍스트를 클릭하면 직원 스케줄 관리 창이 전체 화면으로 열립니다.

열기 방법

헤더 중앙의 날짜/시간에 마우스를 올리면 밑줄이 생깁니다. 클릭 한 번으로 스케줄 창이 열립니다.

▲
날짜/시간 클릭 → 스케줄 표 열기
날짜/시간 텍스트는 클릭 가능한 버튼입니다. 마우스를 올리면 밑줄이 생기며 클릭 시 스케줄 창이 바로 열립니다.

스케줄 표 기능

1
월간 / 주간 보기 전환 — 상단 버튼으로 전환하여 원하는 범위로 확인
2
탭 전환 — 고객상담팀·고객관리팀 등 부서별 스케줄을 탭으로 분리 관리
3
셀 클릭 / 드래그 입력 — 휴무·연차·반차·17시 등 근무 유형을 직접 입력, 같은 버튼 다시 클릭하면 삭제
4
하단 합계 — 직원별 월 근무일수 자동 집계 표시
★ 부관리자 이상 스케줄 수정 권한은 관리자·부관리자에게만 있습니다. 일반 사용자는 조회만 가능합니다.

📂 파일 매뉴얼 ★ 부관리자 이상

업무 매뉴얼 파일(PDF)을 업로드하고 항목별로 등록해두면 챗봇 검색에서 바로 꺼내볼 수 있습니다. 헤더의 📂 파일매뉴얼 버튼으로 열 수 있습니다.

📂 파일매뉴얼 클릭
→
파일 매뉴얼 창
→
매뉴얼 관리 / 파일 목록
관리자·부관리자는 업로드·수정·삭제 가능. 일반 사용자는 열람만 가능합니다.

탭 구성

■ 매뉴얼 관리
PDF를 이미지로 변환하고 항목을 등록·수정·삭제합니다. 챗봇 검색 인덱스 동기화도 여기서 합니다.
■ 파일 목록
업로드된 PDF 파일 목록을 확인하고 다운로드하거나 삭제할 수 있습니다.

플랫폼 선택

매뉴얼은 매니아와 베이 각각 따로 관리됩니다. 상단의 🚀 매니아 / 🚀 베이 버튼으로 전환합니다.

② 항목 수정

항목 수정 탭으로 전환하면 이름·키워드로 검색할 수 있습니다. 항목 클릭 시 수정 폼이 펼쳐지고 수정 또는 삭제 할 수 있습니다.

③ 챗봇 검색 인덱스 동기화

항목을 등록·수정한 뒤 챗봇 검색에 바로 반영되지 않을 때 사용합니다. 검색 인덱스 전체 동기화 버튼을 누르면 등록된 모든 항목이 챗봇 검색에 반영됩니다.

⚠ 매뉴얼 전체 초기화 (DB + 이미지) 버튼은 등록된 항목과 이미지를 전부 삭제합니다. 실수로 누르지 않도록 주의하세요.

❓ FAQ

Q. 봇이 감지를 안 해요
① IMI BOT 프로그램이 실행 중인지 확인 (트레이 아이콘 확인)
② 모니터링 창에서 규칙에 체크가 되어 있는지 확인
③ 봇 상태가 ● 감시 중 인지 확인
④ IMI BOT 프로그램을 종료 후 재시작해 보세요
Q. 알림 팝업이 안 떠요
IMI PRO 프로그램이 실행 중인지 확인하세요. 알림은 프로그램이 켜져 있으면 자동으로 발송됩니다.
그래도 안 뜨면: Windows 설정 → 시스템 → 알림 및 작업 → IMI PRO → 알림 허용 확인
Q. 실수로 필터 제외했어요
📋 로그 → 3 사기글제외 또는 4 비거래제외 탭 → 해당 항목의 ↩ 제외 해제 버튼 클릭
Q. 비밀번호를 잊었어요
로그인 화면 → 비밀번호 찾기 → 이름·사번 입력 → 임시 비밀번호 발급
또는 관리자에게 비밀번호 초기화 요청
Q. 키워드에 콤마(,)가 포함된 단어를 감지하고 싶어요
콤마는 키워드 구분자로 사용되므로 키워드 자체에는 넣을 수 없습니다.
대신 콤마 없이 9검67셋으로 등록하면 9검6,7셋 형태도 자동으로 감지됩니다.
Q. 프로그램 업데이트 후 적용이 안 돼요
최신 버전 설치 파일(IMI PRO Setup.exe)을 실행하여 재설치하세요. 기존 프로그램 위에 덮어쓰기로 설치하면 됩니다.
