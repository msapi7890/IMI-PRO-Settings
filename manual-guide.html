📚 매뉴얼 탭

상단 파란색 버튼. 클릭 시 매뉴얼 검색 모드로 전환됩니다.

④
고객 예상 QNA 탭

AI 기반 고객 예상 질문 생성 기능. 매뉴얼 탭과 별개로 운영됩니다.

⑤
🔍 검색 입력창

여기에 키워드를 입력하고 Enter를 누르면 매뉴얼을 검색합니다. (예: 환불, 출금, 네이버)

③

매칭된 항목이 버튼 형태로 나열됩니다. 원하는 항목을 클릭하면 내용이 펼쳐집니다.
예: ▶ 네이버 아이디 로그인 / ▶ 네이버페이 결제 / ▶ 네이버페이 충전 …

검색창 — 다시 검색할 때 사용

💡
검색이 안 되면 키워드를 더 짧게 줄여보세요.
"네이버페이결제" → "네이버" 또는 "결제"로 검색하는 게 더 잘 찾힙니다.

선택된 항목명

내가 선택한 항목이 우측 상단에 버튼 형태로 표시됩니다.

②
매뉴얼 내용 (표 형식)

제목, 경로, 내용, 수수료, PG사, 한도, 이용, 인증방식 등 항목별 정책이 표로 표시됩니다.
이미지가 아닌 텍스트 데이터로 저장된 항목은 이렇게 표 형식으로 보입니다.

③
🔑 키워드 입력란

이 항목을 더 잘 찾을 수 있도록 추가 키워드를 입력하는 공간입니다.
입력 후 저장하면 다음 번 검색부터 반영됩니다.
키워드를 여러 개 입력할 때 쉼표(,)로 구분합니다. (예: 네이버, 네이버페이, 간편결제)

④
관련 공지 목록

해당 항목과 관련된 최근 공지사항이 자동으로 연결되어 표시됩니다. 클릭 시 공지 내용 확인 가능.

⑤
📝 답변 템플릿 생성

클릭하면 이 항목 내용을 기반으로 고객 답변 초안을 AI가 자동 생성해 줍니다.

화면 4 — 이미지 기반 매뉴얼 (여러 페이지 동시 표시)
①

③

항목 제목

검색으로 찾은 항목명이 표시됩니다. (예: 환급/수수료)

②
매뉴얼 이미지 — 여러 페이지 한 번에 표시

PDF에서 저장된 이미지 페이지들이 나란히 표시됩니다.
등록 시 설정한 시작~끝 페이지 범위가 모두 보입니다. 스크롤로 확인하세요.

③
🔑 키워드 입력란 (이미지 하단)

이미지 기반 항목에도 키워드 입력란이 동일하게 있습니다.
쉼표(,)로 구분해서 검색어를 추가 입력 후 저장하세요.

④
관련 공지

항목과 연관된 공지가 날짜 순으로 표시됩니다. 최신 정책 변경 내용을 확인할 수 있습니다.

💡
표 방식 vs 이미지 방식
텍스트 데이터로 등록된 항목 → 표 형식 (화면 3)
PDF 이미지로 등록된 항목 → 이미지 페이지 나열 (화면 4)
둘 다 키워드 입력란과 관련 공지는 동일하게 표시됩니다.

6

키워드 입력 방법 👑 권한 필요

검색이 잘 되도록 동의어·관련어를 등록합니다
📍 키워드 입력란 위치

매뉴얼 항목을 클릭하면 내용 아래쪽에 키워드 입력란이 나타납니다.
화면 3번 ③ 위치 — "이 항목에 등록된 키워드 검색 가능" 바로 위 입력창

✏️ 입력 방법
🔑 키워드를 입력하세요...   네이버, 네이버페이, 간편결제, 온라인결제
  • 키워드는 쉼표(,)로 구분합니다
  • 공백은 무시되므로 네이버, 네이버페이 와 네이버,네이버페이 동일
  • 항목명과 다른 표현을 위주로 등록하세요 (동의어, 약칭, 오타 등)
  • 입력 후 저장 버튼을 클릭해야 반영됩니다
💡 키워드 등록 예시
항목명: 은행계좌출금
🔑 출금, 정산, 계좌이체, 입금, 무통장, 통장, 계좌번호
항목명: 네이버페이 결제
🔑 네이버, 간편결제, 온라인결제, 네이버페이, 포인트
항목명: 휴대폰 본인인증
🔑 인증, 본인확인, 핸드폰인증, 휴대폰, SKT, KT, LG
⚠️ 키워드 등록 후 검색이 안 될 때
  • 저장 후에도 검색이 안 되면 🔄 검색 인덱스 동기화 버튼을 실행하세요
  • 위치: 매뉴얼 관리 탭 → 하단 동기화 버튼 (관리자 전용)
  • 동기화 후 페이지 새로고침(F5)하면 바로 반영됩니다
✅ 전체 사용 흐름 요약
① 매뉴얼 탭 클릭 → ② 사이트 선택 → ③ 키워드 입력 + Enter → ④ 결과 항목 클릭 → ⑤ 내용 확인 → ⑥ (선택) 답변 템플릿 생성
